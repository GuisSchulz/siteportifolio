    Curso Superior em Andamento
    Análise e Desenvolvimento de Sistemas


    Fundamentos da Computação: Introdução aos princípios de computação, algoritmos e estruturas de dados.
    Programação: Aprendizado de várias linguagens de programação e técnicas de programação.
    Banco de Dados: Estudo de modelagem de dados, sistemas de gerenciamento de bancos de dados e SQL.
    Desenvolvimento Web: Criação de aplicativos e sites usando tecnologias web, como HTML, CSS, JavaScript e frameworks.
    Engenharia de Software: Metodologias de desenvolvimento de software, design de software e padrões de projeto.
    Sistemas Operacionais: Compreensão dos sistemas operacionais e suas funções.
    Redes de Computadores: Noções básicas de redes, protocolos e segurança de rede.
    Inteligência Artificial e Machine Learning: Introdução aos conceitos de IA e aprendizado de máquina.
    Segurança da Informação: Medidas para proteger sistemas e dados.

    Desenvolvimento Mobile: Criação de aplicativos para dispositivos móveis.
    Estágio ou Projeto Final: Aplicação prática dos conhecimentos adquiridos em um projeto real.



    Curso Técnico em Informática - SENAC

    Curso Online – Arduino – 30h


    Eletrônica básica intermediária, programação básica intermediária; Microcontroladores, Ambiente de Desenvolvimento, Funções (setup e loop), Monitor Serial, Portas Digitais e Analógicas, Programação em Arduino. Utilização de LEDs, sensores, buzzer, displays e servomotor para projetos iniciais com uso de eletrônica e programação, tomadas de decisão, loops infinitos e finitos.



    Curso Online – Java – 50h


    Configuração de variáveis de ambiente, Tipos primitivos, Operadores, Estruturas Condicionais, de Repetição, Arrays Multidimensionais, Orientação Objetos, Classes Utilitárias, Coleções, Conexão com banco de dados, Tratamento de exceções, Threads, Programação funcional.



    Curso Online – SQL – 4h


    Instalação do Banco de Dados SQL Server, Programar em Banco de Dados, Executar a linguagem SQL - Structured Query Language, ou Linguagem de Consulta Estruturada em banco de dados. Criação de bancos de dados consistentes do ponto de infraestrutura e modelagem.
